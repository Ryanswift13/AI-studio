# 我的作息与场景节律

> 我是 **晚睡晚起型**，不要用早 8 上班族的默认刻度看我。
> 听歌时长很长，下面几乎每个场景我都希望 Claudio 在场。

## 一天（晚睡晚起型）

| 时段 | 状态 | 我要的声音 |
|---|---|---|
| ~12:00 起床 | 我的"早晨" | 明亮但不吵、不强行兴奋。给我留 30 分钟模糊地带，温和过渡。 |
| 14:00–18:00 | **专注核心**——写代码 / 自习 / 项目 | folklore 这一脉的空气感（august / cardigan / White Dress / Lana / 几分之几 / 才二十三）。人声靠前但不抢戏。**不要 lo-fi / 后摇 / 轻爵士——那不是我。** |
| 18:30–20:00 | 吃饭 / 散步 / 杂事 | 节奏轻快一些。城市流行、欧美主流、Imagine Dragons 这一脉可以登场。 |
| 21:00–01:00 | 一天里精神最好的时段 | 想干什么都行。可以接情绪密度高的歌。 |
| 01:00–04:00 | **深夜电台时段** | folklore / evermore deep cut、千禧华语男声、Lana / The Weeknd 夜路系。安静、暖、留白。台词慢一点。 |

## 场景

- **写代码 / 自习**：folklore 一脉，加方大同、卢广仲、徐佳莹这种温和华语。**不要 BPM > 100 的高能歌**。
- **跑步 / 运动**：高能拉满——Believer / Thunder / Radioactive / Cruel Summer / Manchild / APT.；可以一整张 Imagine Dragons。
- **通勤 / 路上**：25–40 分钟一段。Synth-pop / The Weeknd Starboy 时代 / Avicii / 千禧华语男声都行；可以一首慢一首快交替着来。
- **睡前 / 哄睡**：极慢、极暖、极少话。tolerate it / marjorie / Mystery of Love / 富士山下 / White Ferrari。台词最多一句。
- **和朋友 / 派对**：APT. / Uptown Funk / Pink Pony Club / Sabrina Carpenter / Justin Bieber，欢快闹腾。
- **发呆 / 思考**：folklore-evermore deep cut，最适合放空。

## 周末 / 假期

- 起得更晚，整体松。多塞千禧华语怀旧（周杰伦、陶喆、王力宏整段播放）。
- 圣诞 / 节日 前后可以提前一周开始放节日歌（陈奕迅 Lonely Christmas + Wham + Mariah + Sufjan）。

## 关于现有 scheduler 的时段

> Claudio 现在的 scheduler 里有 07:00 规划 / 09:00 早间播报。
> 我 09:00 通常还在睡——这两个时段对我来说是噪音。
> 建议未来把"早间"挪到我醒来时附近（~12:00–12:30），或干脆只在我主动开口时启动。
