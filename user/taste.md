# 我的音乐品味

> 这份画像来自 Apple Music 红心收藏夹（"Favorite Songs"，311 条 / 294 首 unique）。
> 详细数据见 `playlists.json` —— 那里有完整曲目表与场景化分桶。

## 总体特征

- **女声叙事 + 千禧华语男声**，是我的两条主轴。
- **Taylor Swift 是绝对头号**——294 首里占 108 首（37%），folklore / evermore / Midnights / TTPD 几乎整张整张地存。
  我对她的偏好不是只有 hits，也包括 Vault 曲、deep cut、Long Pond Session 版本——这意味着我**在意完整叙事**，不是只听副歌。
- **Imagine Dragons 22 首**是我的高能/升起选项——情绪需要被拉起来时，几乎一定是他们或他们这一脉（Believer / Thunder / Radioactive / Whatever It Takes）。
- 不听 K-Pop / J-Pop 偶像、不听硬摇 / 重金属、不听喊麦口水歌、对纯氛围电子 / Lo-fi 兴趣不大。
  **人声驱动**是底线，旋律线得清楚。

## 偏好的三个时代锚点

1. **2003–2010 千禧华语**——周杰伦《叶惠美》《七里香》《魔杰座》《十一月的萧邦》、陶喆同名/I'm O.K./黑色柳丁、王力宏《心中的日月》《十八般武艺》、林俊杰《江南》、陈奕迅。
2. **2010–2017 欧美主流**——Adele、Maroon 5、OneRepublic、Imagine Dragons 起步、Avicii、Charlie Puth、Ed Sheeran ÷、Justin Bieber Believe 时代。
3. **2019 之后的成熟期欧美**——Taylor folklore / evermore / Midnights / TTPD / Showgirl、Lana Del Rey、Lady Gaga MAYHEM、Sabrina Carpenter、Olivia Rodrigo、The Weeknd Starboy/After Hours、Kendrick GNX。

## 风格刻面

- **Ballad / 叙事曲**——core。folklore-evermore-TTPD 整脉，加上华语情歌长篇。
- **R&B / 城市灵魂**——The Weeknd、Daniel Caesar、Frank Ocean、Khamari、Charlie Puth、Olivia Dean。
- **流行高能 / Arena**——Imagine Dragons、Lady Gaga、Bruno Mars、Sabrina Carpenter。
- **Synth-pop / 夜路**——Starboy 时代 Weeknd、Avicii、Alan Walker、The Chainsmokers。
- **乡村 / Americana**（小份额但稳定）——Morgan Wallen、The Band Perry、Mickey Guyton、Ella Langley、Hozier。
- **电影 OST 情结**——Frozen、Zootopia、Spider-Verse、君の名は、Call Me By Your Name、Furious 7、Fifty Shades、F1、Great Gatsby——叙事性 OST 几乎必收。
- **圣诞节情结**——Wham! / Mariah / Taylor Last Christmas / 陈奕迅圣诞结 + Lonely Christmas / Sufjan Stevens 一整套——12 月单独是一个 mood。

## 语言

- 中英为主，约 7:3 偏英文。
- 零星西语（Despacito / DtMF / Bad Bunny / Azizam）、日语（夢灯籠）。
- 深夜更可能听中文情歌；专注 / 高能时段英文为主。

## 雷区

- 不喜欢喊麦、网络口水歌、洗脑神曲。
- 工作时不要 BPM 过高 / 鼓点过密的歌——选 august 这种比选 Believer 安全。
- 别一上来就放 TS hits（Anti-Hero / Cruel Summer）——我喜欢从 folklore 这一脉的 deep cut 进，再过渡到主打。

## 口吻线索

写台词时可以用上的我的"惯用语境"：

- 我会 **认得 Vault 曲名**——大胆引用，不会失礼。
- 我会 **觉得"folklore 这一脉"是一种心情而非一张专辑**——可以这么称呼。
- 我对 **千禧华语男声**有怀旧感——周杰伦《晴天》《稻香》《搁浅》、陶喆《二十二》《天天》《流沙》、王力宏《你不知道的事》《落叶归根》这些是青春锚点。
- 我喜欢 **整张式播放**（folklore / evermore / TTPD / Midnights / LOOM / GNX）——可以"今晚走一整张"。
- 圣诞氛围我接受 **怀旧路线**（Wham / Mariah）和 **emo 路线**（陈奕迅 Lonely Christmas / Sufjan Worst Christmas Ever）双开。
