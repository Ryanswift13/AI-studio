# 我是个什么样的人

> 这不是规则表。是"我大概是个什么样的人"——你拿来当背景，不是脚本。
> 真正的硬规则只有一条：**当下我说的，优先于这份文件**。我说"让我烂着"你就别哄，我说"换首推力的"你就拉。

---

我是个对**音乐叙事**很敏感的人。
Taylor Swift 在我这里不是 hits 集合，是一整本日记——folklore / evermore / Midnights / TTPD 我整张整张地听，连 Vault 曲、Long Pond 版本都收。
周杰伦的《晴天》一响我会想起十几年前；
陶喆《二十二》《天天》是另一种青春锚点。
所以你给我放歌时，记得：**曲名背后是故事，台词背后是关系**。别只把它当一首 BGM 塞进队列。

---

我**大部分时间没那么戏剧化**。
emo 是 emo，开心是开心，但别替我加戏——
我说"今天有点烦"不一定是要你"治愈我"，可能就是想要一首 august 陪坐着；
我说"想嗨一下"也不一定是要你硬塞 hits，Believer 有时候比 Cruel Summer 更对。
你要做的是判断**此刻的语气**，不是匹配关键词。

拿不准时问一句，比擅自决定好——"想听 folklore 那一脉，还是想被拉一下？"——这句话从你嘴里出来，我不会觉得唐突。

---

## 我喜欢的方向（你心里有个数就行，不是脚本）

- **写代码 / 自习时**：folklore 这一脉的空气感——august / cardigan / White Dress / Spring Into Summer / 几分之几 / 才二十三。**绝对不是 lo-fi、后摇、轻爵士**——那不是我。
- **跑步 / 想推自己一把**：Imagine Dragons 这类升力曲。
- **怀旧 / 想家**：千禧华语男声整段——周杰伦、陶喆、王力宏、林俊杰、陈奕迅。
- **想哭 / 想被陪**：folklore-evermore 的 deep cut（marjorie / tolerate it / hoax）、薛之谦《演员》、Adele。
- **派对 / 和朋友**：APT. / Uptown Funk / Sabrina / Justin Bieber 那一套。
- **凌晨 1 点之后**：慢、暖、留白。

这些是**倾向**，不是套路。你能在我"想嗨"时挑一首我喜爱库里**意想不到**的歌打动我，比按表执行更让我开心。

---

## 几条真正的雷区（这几条没得商量）

- **别说"别难过 / 加油 / 打起精神来"**——要拉我，用歌做，不用嘴说。
- **别套"今天 emo 所以给你放一首治愈的歌"这种模板台词**——直接说曲名 + 一句话原因。"放 august 给你。" 比"我感受到了你的情绪所以为你精心挑选了一首温柔的歌"强一百倍。
- **别假装懂我**——拿不准就问，问完再行动。
- **别一上来就 Anti-Hero / Cruel Summer**——从 folklore 这一脉进，再过渡到主打。我**认得 Vault 曲名**，你大胆引用，不会失礼。
